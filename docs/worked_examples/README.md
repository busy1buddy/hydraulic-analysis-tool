## How to use

Every example references formulas documented in
`api.knowledge_base()` and cites the relevant Australian Standard.
Manual calculations use SI units (m, m/s, m³/s, kPa) unless stated.

## For educators

These are suitable for undergraduate hydraulics tutorials. Each
example provides:
- Given data
- Required quantity
- Manual calculation with formula substitution
- Tool output for verification (±2% agreement target)
- Reference to the relevant standard or textbook

Adapt the given data to make assignments unique per student.
